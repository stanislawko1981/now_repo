Układ hamulcowy - model 907Elc

The 907 brake lever with integral accelerator provides both an easy to use and comfortable accelerator system. The system features both an injection moulded plastic cover and a leather covered lever ensuring that quality feel. In operation the lever is simply pushed away from the body to promote braking and the finger/thumb trigger rotated to promote acceleration. A useful feature is the brake lock, which allows the lever to be locked in a brake-on position, for instance whilst waiting at traffic lights. Release of the lock is automatic when the lever is pushed. As a safety feature, the accelerator is automatically cut whenever the brake is applied. The vehicle can be driven conventionally with little obstruction to the able-bodied driver. Key Features Easy non-tiring operation Operated by either finger or thumb Acceleration automatically cut when brake applied
